Machine Learning Approach to Classify Precipitation Type from A Passive Microwave Sensor
Spandan Das, Jie Gong, Chenxi Wang, Dong Wu, Stephen Munchak, William Olson
American Geophysical Union (AGU) Fall Meeting, 2020
abstract / poster

Using collocated precipitation flags from the Global Precipitation Measurement's (GPM) Dual-Frequency Precipitation Radar (DPR) as the “truth”, this project employs machine learning models to train and test the predictability and accuracy of using passive microwave-only observations together with ancillary atmosphere information from reanalysis.
